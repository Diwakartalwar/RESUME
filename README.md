# RESUME
a basic 9 module resume can be edited and used as a web reume made with django can be switched into flat files with django distill or django bakery 

A minimal Django project providing **two pre-designed resume templates** that can be easily edited and deployed as a personal resume website.

Instead of building resumes dynamically, this project focuses on **clean, ready-to-use layouts** that you can customize with your own data.

---

## 🚀 Features

- 📄 Two pre-built resume templates
- 🎨 Clean and minimal design
- ⚡ Django-powered rendering
- 🛠️ Easy content editing (no complex setup)
- 🌐 Can be converted into static HTML

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS
